# Test Assets

Place a sample WAV file here for testing STT:

```
tests/assets/sample.wav
```

**Requirements:**
- Format: WAV
- Sample rate: 16000 Hz
- Channels: 1 (mono)
- Sample audio: Any spoken audio (5-10 seconds recommended)

You can record your own or use an online tool like [ttsmaker.com](https://ttsmaker.com/) to generate test audio.
